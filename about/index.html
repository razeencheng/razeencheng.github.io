建站历史

2016年 开始了写博客，中间折腾过很多博客平台。

  • 最早使用过Wordpress

    • 优点：操作简单，皮肤多，插件也不少。
    • 缺点：PHP+MySSL我有点不喜欢，而且总是会抛出一些奇奇怪怪的问题，弃之。

  • 之后就是Hexo

    • 优点：快速部署，0成本维护，社区活跃。
    • 缺点：没有好用的后台，写文章步骤较多。
    • 相关博客： 云服务器搭建hexo博客，git hooks自动更新 GitHub Pages自定义域名开启HTTPS

  • 中间还折腾过ghost

    • 优点：部署快速，自定义页面简单，后台丰富。
    • 缺点：toc/归档等折腾起来比较费力气。
    • 相关博客：开始使用Ghost Ghost主题收集

  • 然后就是EIBlog. eiblog是朋友用go开发的,也是我用的时间最长的一个博客。

    • 优点：速度特快，简洁，有独立后台，书写方便。
    • 缺点：自定义页面操作比较复杂。

  • 2021 年初，又换成了Hexo + Next。

    • 相关博客：我又又又把博客迁移了

  • 现在因为对 Golang 的偏爱，又切换到了 Hugo + Hermit。

为什么写博客

很多时间我们会因为出发了太久而忘了最初的目的。

在这里写下我写博客的初衷，告诉自己，不忘初心。

  1. 输出学到的知识，加以巩固；
  2. 有一个自己自由发声的平台，我的思想，我的计划；
  3. 希望将我学到的知识，踩到的坑分享给他人，帮助他人；
  4. 提升个人表达；
  5. 争取一定流量，最后能有一定的收入；
  6. 结识一些共同爱好的好友。
